# AnimalZoo

AnimalZoo 是一个使用 Vue 3 + Vite 构建的前端示例/小项目，展示动物相关的 UI 与交互组件（图片库、卡片展示、筛选、详情页等）。本仓库包含基本的项目结构、开发脚本与构建配置，适合作为学习或演示用途。

## 主要特性
- 基于 Vue 3 与 Vite 的轻量前端项目模板
- 简单的动物列表与详情展示页面
- 本地开发服务器与生产构建脚本

## 快速开始
先确保已安装 Node.js（推荐 18.x 或更高）和 npm/yarn。

1. 克隆仓库：

   git clone https://github.com/Token2049/AnimalZoo.git
   cd AnimalZoo

2. 安装依赖：

   npm install
   # 或者
   # yarn

3. 启动开发服务器：

   npm run dev

4. 生产构建：

   npm run build

## 项目结构（简要）
- public/ — 静态资源
- src/ — 源代码（组件、样式、路由等）
- index.html — 应用入口
- vite.config.js — Vite 配置

## 常用脚本（package.json）
- npm run dev — 启动开发服务器
- npm run build — 构建生产包
- npm run preview — 预览构建结果（如果配置）
- npm test — 运行测试（如果存在）
